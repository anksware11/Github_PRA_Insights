# Privacy Policy

Last Updated: March 2026

1. Data Collection: The AI PR Analyzer does not collect or store your personal data. We only access GitHub Pull Request text and metadata via the GitHub API to provide analysis.
2. Data Usage: Your code is processed in real-time to generate review insights. We do not use your code to train our AI models.
3. Third Parties: Data is sent securely to our AI processing partner for the sole purpose of generating the analysis. No data is sold or shared with advertisers.
4. Permissions: The extension requires "Tabs" and "Storage" permissions to function correctly and save your local preferences.
